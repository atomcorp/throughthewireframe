Through the Wireframe

Summary

Through the wireframe aims to make creating wireframes super simple.

It's just an adapted version of Imperfect Buttons by Tiffany Rayside, so full credit to her.

Box

The simplest use is to add a .box class to an element.

There are 5 predefined styles

.box
.box__2
.box__3
.box__4
.box__5

Round

If you want to highlight something with a more unusual shape use .round

Again, there are 5 predefined styles

.round
.round__2
.round__3
.round__4
.round__5

Image Placeholder

To demonstrate an area where an image would be you can use a background

Use .box .no-border if a background is messing up a containering bordered box
