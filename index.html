Through the Wireframe

Summary

Through the Wireframe aims to make creating wireframes super simple.

It's just an adapted version of Imperfect Buttons by Tiffany Rayside, so full credit to her.

For a complete example of what you can do, look at a facebook wireframe page.

View on Github

Tips

  • Use flex. It's the quickest way to rapidly build prototype layouts. Seeing as it's just prototype who cares about IE? CSS Trick's guide is indispensable if you're unfamiliar with flex.
  • For doing stuff like this, using negative margins to overlap:

    margin: 0 -4px;

  • Use libraries and plugins, don't reinvent the wheel.
